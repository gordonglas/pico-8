# Resources for learning Pico-8

The [cart-snippets](cart-snippets) folder contains simple carts that demonstrate something. Cart files have a `.p8` extension, but can be viewed in plain-text to see it's lua code, or can be loaded into [Pico-8](https://www.lexaloffle.com/pico-8.php).
